# Changelog

## v0.2.6 (2026-06-13)

### 🐛 Bug 修复

- **正式版 i18n 模组词典**：修复安装包内置词典可能被打成 Git LFS 指针文件，导致检查或更新词典时报 `file is not a database` 的问题
- **词典更新自修复**：即使内置词典文件损坏，设置页也会提示需要更新；点击更新会重新下载真实 SQLite 词典并验证后替换

## v0.2.5 (2026-06-13)

### 🔧 改进

- **关于与更新**：关于页先展示产品介绍和项目入口，再集中展示版本与更新操作；检查到新版本时不再出现空进度条

### 🐛 Bug 修复

## v0.2.4 (2026-06-13)

### ✨ 新功能

- **全自动模式**：扫描后可自动切换到翻译、校对和打包页面，自动重试失败条目，并在每个阶段展示真实页面状态
- **i18n 模组词典**：内置最新版 i18n 参考词典，设置页支持检查更新和一键更新，用于提升中文翻译参考质量
- **词库管理**：词库页新增清空词库操作，并使用应用内二次确认弹窗防止误删

### 🔧 改进

- **翻译质量**：优化 LLM 提示词、术语参考和占位符保护，减少机翻感、误译和纯英文残留
- **参考词典性能**：优化 i18n 参考词典的加载与查询，避免启动预热和翻译阶段因词典访问卡顿
- **全自动体验**：全自动运行期间会锁定普通操作，提供中断流程按钮，并正确显示侧边栏阶段状态
- **设置默认值**：默认并发数量调整为 100，默认批次大小调整为 80
- **应用信息**：设置页关于部分增加项目简介、项目原址和作者主页入口
- **资源包信息**：生成资源包的元数据会显示生成时间，并优化资源包描述文案

### 🐛 Bug 修复

- **JSON 语言文件兼容**：空 JSON 语言文件不再误报解析失败，UTF-16 编码语言文件可以正常读取
- **打包后打开目录**：修复 Windows 长路径被转换为 `//?/E:/...` 后无法打开资源包目录的问题
- **预热动画**：修复预热阶段勾选动画偶发卡住或播放不完整的问题
- **更新检查按钮**：修复检查更新时按钮消失导致布局异常的问题
- **全自动导航状态**：修复全自动流程中侧边栏阶段没有正确显示进行中和已完成的问题

## v0.1.0 (2026-06-11)

### 🎉 首个公开发布

Aaalice Minecraft Translator 的第一个公开发布版本。整合包汉化工具，通过 LLM 批量翻译模组文本并输出为资源包。

### ✨ 新功能

- **翻译管线**：完整的 5 阶段翻译流水线（扫描 → 词典匹配 → CFPA 参考 → LLM 翻译 → 打包）
- **增量扫描**：支持重复扫描时增量更新，避免重复工作
- **LLM 翻译**：智能分批次、并发请求、占位符保护（Shield）、429 自动降速
- **断点续翻**：支持中断后继续翻译，已有结果不重复请求
- **词典系统**：SQLite 持久化词典 + 模糊匹配 + 手动词条管理
- **CFPA 参考**：集成 CFPA 中文翻译资源复用
- **资源包打包**：自动生成 Minecraft 资源包，支持 pack.png 图标嵌入
- **人工校对**：校对工作台支持逐条编辑、单条目重翻
- **重试机制**：翻译失败条目一键重试，不覆盖已有结果
- **主题系统**：暗夜模式切换 + CSS 圆形遮罩动画（startViewTransition）
- **多语言 UI**：支持 5 种界面语言（简体中文 / English / 日本語 / 한국어 / Русский）
- **实时日志**：翻译过程实时日志面板 + 日志过滤/搜索
- **扫描进度**：并行 jar 扫描引擎 + 可视化进度条
- **设置页面**：完整的 LLM API 配置（供应商分组、参数调节、自定义 prompt）
- **字体选择**：自动枚举系统字体，界面字体即时预览
- **资源包复用**：自动检测并利用已有的资源包翻译
- **侧边栏导航**：三态导航状态（idle / busy / completed）
- **流程控制**：顶栏流程状态指示，限制翻译完成前不得打包
- **启动屏**：SplashScreen 加载动画 + 后台预热

### 🔧 技术亮点

- Tauri 2 + Rust 后端，React 18 + TypeScript 前端
- Rayon 并行 jar 扫描（4 线程池限制）
- reqwest 连接池复用 + 并发工作池
- Zustand 全局状态（从 useReducer 渐进迁移中）
- 虚拟滚动表格（react-virtuoso）支撑千级条目
- 组件复用体系（DataTable / SearchInput / SortableTableHeader）
- 76+9 单元/集成测试覆盖核心逻辑
- 检查点恢复：LLM 阶段读取已有 JSONL 跳过已翻译条目
