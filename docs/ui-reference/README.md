# UI 参考图

本目录保存逐页生成的 UI 参考图。图片只是视觉和信息架构参考，后续实现时以功能规格和实际组件约束为准。

## 图片清单

### 01 品牌与 Logo

![品牌与 Logo](images/01-brand-logo.png)

用途：

- app icon。
- 侧边栏 wordmark。
- 资源包图标基础方向。

### 02 总览页

![总览页](images/02-dashboard.png)

用途：

- 实例选择。
- 扫描摘要。
- 资源包复用概览。
- 开始翻译入口。

### 03 翻译进度页

![翻译进度页](images/03-translation-progress.png)

用途：

- 任务阶段。
- per-mod 队列。
- 费用和 token。
- 当前 batch 预览。
- 校验警告。

### 04 词典浏览与纠错页

![词典浏览与纠错页](images/04-dictionary-browser.png)

用途：

- 词典搜索。
- 来源优先级。
- 手动纠错。
- 导入导出。

### 05 设置页

![设置页](images/05-settings.png)

用途：

- LLM URL、Key、model。
- 并发和 batch size。
- 资源包复用。
- FTB 开关。
- 日志策略。

### 06 资源包打包确认页

![资源包打包确认页](images/06-packaging-confirmation.png)

用途：

- 输出资源包摘要。
- 冲突确认。
- dry-run diff。
- 复制或替换确认。

### 07 日志诊断页

![日志诊断页](images/07-logs-diagnostics.png)

用途：

- 日志过滤。
- 错误详情。
- retry history。
- 日志导出。

### 08 FTB 任务汉化页

![FTB 任务汉化页](images/08-ftb-quests.png)

用途：

- FTB Quest tree。
- 任务文本翻译。
- 手动纠错。

### 09 硬编码实验室

![硬编码实验室](images/09-hardcoded-lab.png)

用途：

- Vault Patcher 草案。
- 风险确认。
- 二期实验功能入口。

首期不自动应用硬编码补丁。
